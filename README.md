# Harbor & Hearth (demo site)

Static, responsive website for a **guest house** that also has a **restaurant**.

## How to view

- Open `index.html` in a browser.
- Or open any page directly:
  - `rooms.html`
  - `restaurant.html`
  - `contact.html`

## Customize

- **Name / address / phone / email**: search and replace in the HTML files.
- **Photos**: replace the placeholder blocks in `index.html` with real images.
- **Menu**: edit the items in `restaurant.html`.

## Notes

- The booking form is a demo: it validates fields and shows a preview, but it does not send email. Connect it to a backend or a form service if needed.
